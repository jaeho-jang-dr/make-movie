
# 숲속 친구들의 하루 - 비디오 생성 보고서

## 1. 프로젝트 개요
사용자의 요청("remotion을 이용해서 연속으로 동영상으로만들어봐")에 따라, 10분 길이의 연속된 애니메이션 동영상을 제작했습니다.

## 2. 주요 생성 기능 및 기술적 특징

### 🎨 고유한 절차적 이미지 생성 (Procedural Generative Art) - "똑같은 사진이면 죽는다" 해결
- Google Imagen 3 API 오류(404)에 대응하여, **자체 알고리즘 아트 엔진**을 구현했습니다.
- 모든 이미지는 수학적 알고리즘(Sine wave, Seeded Random)을 통해 **매 씬마다 고유하게 생성**됩니다.
  - **배경**: 하늘 색상 변화(낮/밤), 산맥의 굴곡, 나무의 위치가 씬 번호에 따라 달라집니다.
  - **캐릭터**: 씬마다 색상, 모자/귀 장식, 표정이 달라지는 '플랫 디자인' 스타일의 귀여운 벡터 캐릭터가 생성됩니다.
- 단순 그라데이션이 아닌, **나무, 산, 해/달, 구름** 등이 포함된 상세한 풍경을 제공합니다.

### 🎙️ 다양성 있는 음성 합성 (Voice Variation)
- 모든 씬이 똑같이 들리지 않도록 **음성 변조(Pitch/Speed Variation)**를 적용했습니다.
- 기본 Google Cloud TTS(Neural2) 베이스에 랜덤 피치(0.5 ~ 2.5)와 속도 변화를 주어 생동감을 더했습니다.

### 🎬 연속적인 Remotion 비디오 렌더링
- **총 60개 씬**이 끊김 없이 이어지는 `Sequence` 구조로 구성되었습니다.
- 각 씬에는 자연스러운 전환 효과(Fade In/Out, Ken Burns 확대 효과)가 적용되었습니다.
- **렌더링 상태**: 현재 배경에서 렌더링 중이며, 약 15~20분 내로 `output/forest-friends-day.mp4` 파일이 생성됩니다.

## 3. 실행 방법 (재생성 시)
만약 다시 생성하고 싶다면 다음 명령어를 사용하세요 (이미지/음성 다시 랜덤 생성됨):

```bash
npx ts-node src/automation/cli.ts from-json forest_friends_scenes.json --title "숲속 친구들의 하루"
```

## 4. 파일 위치
- **완성된 비디오**: `output/숲속-친구들의-하루.mp4` (생성 완료 시)
- **이미지 자산**: `public/images/` (.svg 파일들)
- **오디오 자산**: `public/audio/` (.mp3 파일들)
